34 Moto Kurye Istanbul - cPanel Node.js App Yukleme

1. cPanel > Setup Node.js App ac.
2. Node.js surumu olarak 20 sec.
3. Application root olarak ornegin /home/u2629744/34motokuryeistanbul sec.
4. Startup file olarak app.js sec.
5. Bu klasore zip icerigini yukle ve cikar.
6. Environment Variables alanina .env.example icindeki degerleri gir.
7. NPM Install gerekiyorsa calistir.
8. App'i restart et.
9. Domaini uygulamaya bagla.

Notlar:
- server.js standalone Next.js server dosyasidir.
- .next/static ve public klasorleri ayni yapida korunmalidir.
- SMTP bilgileri girilmezse teklif formu WhatsApp/mailto fallback ile calisir.
